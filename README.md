# 9TeamTD 실행 안내

## 실행
Releases에서 다운로드 후
- `9TeamTd.exe` 실행

## 조작
- 이동: `W A S D`
- 메뉴/닫기: `ESC`

## 문제 해결
- 입력이 안 먹으면: 게임 화면을 한 번 클릭해서 포커스를 잡으세요.
- 실행이 안 되면: 압축을 풀고 **폴더째로** 실행하세요. (파일 일부만 옮기면 실행이 깨질 수 있음)

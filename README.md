# 9TeamTD 빌드 실행 안내

이 폴더는 게임 실행 파일(빌드)입니다. 아래 순서대로 실행하세요.

---

## 1) 실행 방법
1. `9TeamTd.exe` 더블클릭
2. 실행이 안 되면 아래 “문제 해결”을 확인하세요.

---

## 2) 권장 실행 환경
- OS: Windows 10/11 (64-bit)
- 입력: 키보드/마우스
- 해상도: 1920x1080 권장 (창모드/전체화면은 게임 내 설정에 따름)

---

## 3) 조작 방법 (기본)
- 이동: `W A S D`
- (있다면) 메뉴/닫기: `ESC`

> 조작이 안 되면: 게임 화면을 한 번 클릭해서 포커스를 잡고 다시 입력하세요.

---

## 4) 저장/설정 파일 위치 (있을 때만)
- 게임이 설정/세이브를 만든다면 보통 다음 경로 중 하나에 생성됩니다.
  - `C:\Users\<User>\AppData\LocalLow\<CompanyName>\<ProductName>\`
  - `C:\Users\<User>\AppData\Local\<CompanyName>\<ProductName>\`

※ 프로젝트 설정에 따라 위치는 달라질 수 있습니다.

---

## 5) 문제 해결 (자주 나오는 것들)

### Q1. 실행하면 바로 꺼져요 / 검은 화면이에요
- 압축으로 받았다면 **폴더째로 압축 해제** 후 실행했는지 확인
- 폴더 안 파일을 일부만 옮기면 실행이 깨질 수 있습니다.
- 그래픽 드라이버 업데이트 후 재시도

### Q2. “VCRUNTIME / MSVCP***.dll이 없다” 같은 오류가 떠요
- **Microsoft Visual C++ 재배포 패키지(2015~2022, x64)** 설치 후 재실행

### Q3. Windows에서 “알 수 없는 게시자” 경고가 떠요
- [추가 정보] → [실행]으로 진행
- 사내/학원 PC면 보안 정책으로 차단될 수 있습니다.

### Q4. 입력(WASD)이 안 먹어요
- 게임 화면을 클릭해서 포커스 잡기
- 한/영키 전환 후 재시도
- 다른 프로그램(채팅/브라우저)이 키보드 포커스를 잡고 있지 않은지 확인
